# 启航电商OMS系统 分析报告

## 1. 项目概述
这是一个**开源电商订单管理系统(OMS)**，版本2.0.1，支持淘宝、京东、拼多多、抖店、微信小店等主流电商平台。

## 2. 技术架构

| 层级 | 技术栈 |
|------|--------|
| **后端** | Java 17 + Spring Boot 3.0.2 + MyBatis-Plus |
| **前端** | Vue 2.6 + Element UI + Vuex |
| **数据库** | MySQL 8 + Redis 7 |
| **认证** | Spring Security + JWT |
| **服务器** | Undertow (替代Tomcat) |

## 3. 模块结构

```
qihang-oms/
├── api/          # API接口层 (主启动模块)
├── core/         # 核心模块
│   ├── common/   # 通用工具
│   ├── security/ # 安全认证
│   ├── model/    # 数据模型
│   └── mapper/   # 数据访问
├── model/        # 实体模型
├── mapper/       # MyBatis Mapper
├── service/      # 业务服务层
├── services/     # 平台服务 (淘宝/京东/拼多多/抖店/微信)
├── vue/          # 前端Vue项目
└── docs/         # 文档
```

## 4. 核心功能

| 模块 | 功能 |
|------|------|
| **商品管理** | 商品库、SKU、库存、店铺商品关联、分类、品牌、供应商 |
| **订单管理** | 店铺订单同步、订单审核、发货订单库 |
| **发货管理** | 手动发货、供应商发货、电子面单、备货出库 |
| **售后管理** | 退货退款、补发、换货、订单拦截 |
| **平台对接** | 淘宝/京东/拼多多/抖店/微信小店 API对接 |

## 5. 项目特点
- 开箱即用，完整OMS解决方案
- 支持多电商平台订单统一管理
- 支持供应商发货模式
- 提供API接口，可对接内部ERP
- AGPL-3.0 开源协议

## 6. 部署要求
- Java 17 + Maven 3.8
- Node.js v16.x
- MySQL 8 + Redis 7
- Nginx 反向代理
